# European Countries

Shows European countries with visitor metrics, engagement data, and traffic distribution in a data table format.

## Predefined Filter

This report applies a predefined filter to show only European countries:

| Filter | Value |
|--------|-------|
| `continent_code` | Europe |

This filter is applied automatically and cannot be changed by the user.
