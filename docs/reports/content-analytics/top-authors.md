# Top Authors

Shows all authors with their content performance metrics in a data table format.

## Report Configuration

- **Type**: Report Page (Hidden)
- **Menu Group**: Content Analytics
- **Show in Menu**: No (accessed from Top Authors widget in Authors report)
- **Add-on**: Free (Data Plus required for custom post types)
- **Status**: Done
- **Figma Design**: [Add link when available]

## Access

This page is accessed by clicking "See all authors" link in the [Top Authors](../../widgets/top-authors.md) widget on the [Authors Report](authors.md).

## Add-on Requirements

- **Free**: Posts and Pages
- **Data Plus**: Custom post types

## Available Interactions

- **Date Picker**: Select time period for data
- **Post Type Filter**: Filter by post type (default: Posts)
  - Free: Posts, Pages
  - Data Plus: Custom post types
