# Category Pages

Shows performance metrics for category archive pages (e.g., `/category/technology/`), not individual posts within categories.

## Report Configuration

- **Type**: Report Page
- **Menu Group**: Page Insights
- **Show in Menu**: Yes
- **Add-on**: Free
- **Status**: Done

## Available Interactions
