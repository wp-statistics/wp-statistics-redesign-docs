# Social Media

Shows social media traffic driving visitors to your website, with trend analysis and detailed breakdown by platform.

## Page Configuration

- **Menu Visibility**: Shown in main menu
- **Add-on**: Free
- **Status**: Done
- **Figma Design**: [Add link when available]

## Available Interactions

- **Date Picker**: Select date range for analysis
- **Source Category**: Filter by traffic type (default = All)
  - All: Shows both organic and paid social media traffic
  - Organic Social: Shows only organic social media traffic
  - Paid Social: Shows only paid social media traffic

## Widget Layout

### Row 1 (Full Width)

[Traffic Source Chart](../../widgets/traffic-source-chart.md) - Line chart showing visitor trends from social media platforms with breakdown by top 3 platforms. This widget is filtered to display only social media traffic.

### Row 2 (Full Width)

[Referrers Table](../../widgets/referrers-table.md) - Data table showing detailed metrics for each social media platform. This widget is filtered to display only social media referrers (Facebook, Twitter, LinkedIn, etc.).

**Note**: The Referrers Table widget is reused here but displays only social media traffic, not all referrers.
