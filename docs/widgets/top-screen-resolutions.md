# Top Screen Resolutions Widget

Displays the top 5 screen resolutions by visitor count in a horizontal bar list format with previous period comparison.

## Widget Configuration

- **Component**: [Horizontal Bar List](../components/horizontal-bar-list.md)
- **Add-on**: Free (included in base plugin)
- **Status**: Done
- **Figma Design**: [Add link when available]

## Component Configuration

| Property | Value |
|----------|-------|
| **Max Visible Items** | 5 |
| **Show Previous Period** | ✅ Yes |
| **Show Icons** | ❌ No |
| **Enable Other Item** | ❌ No |
| **Link to Full Report** | ✅ Yes (links to Screen Resolutions report) |

## Used In Reports

- [Devices Overview](../reports/devices/devices-overview.md)

## Empty State

When no screen resolution data is available for the selected period:
- "No data available for the selected period"
