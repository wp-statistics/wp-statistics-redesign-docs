# Top Cities Overview Widget

Displays the top 5 cities by visitor count in a horizontal bar list format.

## Widget Configuration

- **Component**: [Horizontal Bar List](../components/horizontal-bar-list.md)
- **Add-on**: Free (included in base plugin)
- **Status**: Done
- **Figma Design**: [Add link when available]

## Component Configuration

| Property | Value |
|----------|-------|
| **Max Visible Items** | 5 |
| **Show Previous Period** | No |
| **Show Icons** | No |
| **Enable Other Item** | No |
| **Link to Full Report** | Yes (links to Cities report) |

## Used In Reports
