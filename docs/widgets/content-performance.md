# Content Performance Widget

Displays visitor, view, and publishing trends over time for the selected post type.

## Widget Configuration

- **Component**: [Line Chart](../components/line-chart.md)
- **Add-on**: Free (included in base plugin)
- **Status**: Not Started
- **Figma Design**: [Add link when available]

## Component Configuration

| Property | Value |
|----------|-------|
| **Title** | Performance |
| **Primary Metric** | Visitors |
| **Secondary Metric** | Views |
| **Bar Metric** | Published Contents |
| **Previous Period** | ✅ Yes |
| **Metric Toggle** | ✅ Yes |
| **Timeframe Selector** | ✅ Yes |
| **Legend Display** | ✅ Yes |

## Used In Reports

This widget is used in the following reports:

- [Content](../reports/content-analytics/content.md) - Row 2, full width
