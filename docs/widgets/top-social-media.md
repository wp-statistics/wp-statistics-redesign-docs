# Top Social Media Widget

Displays the top 5 social media platforms by referral count in a horizontal bar list format with previous period comparison.

## Widget Configuration

- **Component**: [Horizontal Bar List](../components/horizontal-bar-list.md)
- **Add-on**: Free (included in base plugin)
- **Status**: Done
- **Figma Design**: [Add link when available]

## Component Configuration

| Property | Value |
|----------|-------|
| **Max Visible Items** | 5 |
| **Show Previous Period** | ✅ Yes |
| **Show Icons** | ❌ No |
| **Enable Other Item** | ❌ No |
| **Link to Full Report** | ✅ Yes (links to Social Media report) |

## Used In Reports

This widget is used in the following reports:

- [Referrals Overview](../reports/referrals/referrals-overview.md) - Row 5, right column

## Empty State

When no social media data is available for the selected period:
- "No data available for the selected period"
