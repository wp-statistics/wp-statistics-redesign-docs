# Top European Countries Widget

Displays the top 5 European countries by visitor count in a horizontal bar list format with country flags.

## Widget Configuration

- **Component**: [Horizontal Bar List](../components/horizontal-bar-list.md)
- **Add-on**: Free (included in base plugin)
- **Status**: Done
- **Figma Design**: [Add link when available]

## Component Configuration

| Property | Value |
|----------|-------|
| **Max Visible Items** | 5 |
| **Show Previous Period** | No |
| **Show Icons** | Yes (country flags) |
| **Enable Other Item** | No |
| **Link to Full Report** | Yes (links to European Countries report) |

## Used In Reports

- [Geographic Overview](../reports/geographic/geographic-overview.md)

## Empty State

When no European country data is available for the selected period:
- "No data available for the selected period"
