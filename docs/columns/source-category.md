# Source Category Column

Shows the traffic channel category (e.g., Search Engines, Social Media, Direct, Referral).

- **Type**: Column (Reusable table column)
- **Status**: Done

## Used In Widgets

- [Source Categories Table](../widgets/source-categories-table.md)

## Display

**Label:** Source Category

**Format:**
- Text string
- Left-aligned in table cell
- Displayed as plain text (non-clickable)

**Typical Values:**
- Search Engines
- Social Media
- Direct
- Referral
- Email
- Paid Advertising
- Other

## Sortable Behavior

This column is **not sortable**.
