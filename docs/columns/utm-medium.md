# UTM Medium Column

Shows the UTM medium parameter value from campaign URLs.

## Column Configuration

- **Type**: Column (Reusable table column)
- **Status**: Done
- **Figma Design**: [Add link when available]

## Used In Widgets

This column is used in the following widgets:

- [Campaigns Table](../widgets/campaigns-table.md)

## Display

**Label:** Medium

**Format:**
- Text string
- Left-aligned in table cell
- Displays the value of the `utm_medium` parameter

**Typical Values:**
- cpc (cost per click)
- email
- social
- organic
- referral
- banner

## Sortable Behavior

This column is **not sortable**.
