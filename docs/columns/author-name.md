# Author Name Column

Shows the author name with avatar, linked to the Individual Author report.

## Column Configuration

- **Type**: Column (Reusable table column)
- **Status**: Done
- **Figma Design**: [Add link when available]

## Used In Widgets

This column is used in the following widgets:

- [Top Authors Table](../widgets/top-authors-table.md)

## Display

**Label:** Author

**Format:** Avatar + Author name

**Alignment:** Left

**Avatar:** Author's Gravatar or default placeholder

## Interactive Elements

### Click Action

Opens the [Individual Author Report](../reports/content-analytics/individual-author.md) for this author in the same tab.

### Hover Tooltip

Displays author display name (useful when name is truncated).

## Sortable Behavior

- **Sortable:** Yes
- **Default Sort Order:** Alphabetical (A-Z)
- **Sort Type:** Text
