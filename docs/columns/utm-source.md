# UTM Source Column

Shows the UTM source parameter value from campaign URLs.

## Column Configuration

- **Type**: Column (Reusable table column)
- **Status**: Done
- **Figma Design**: [Add link when available]

## Used In Widgets

This column is used in the following widgets:

- [Campaigns Table](../widgets/campaigns-table.md)

## Display

**Label:** Source

**Format:**
- Text string
- Left-aligned in table cell
- Displays the value of the `utm_source` parameter

**Typical Values:**
- google
- facebook
- newsletter
- twitter
- linkedin
- email

## Sortable Behavior

This column is **not sortable**.
