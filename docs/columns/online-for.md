# Online For Column

Shows the duration a visitor has been active on the website during their current session.

## Column Configuration

- **Type**: Column (Reusable table column)
- **Status**: Done
- **Figma Design**: [Add link when available]

## Used In Widgets

This column is used in the following widgets:

- [Online Visitors Table](../widgets/online-visitors-table.md)

## Display

**Value:** Time duration in HH:MM:SS format

**Format:**
- Hours:Minutes:Seconds
- Zero-padded (e.g., 00:08:34)
- Examples: 00:00:45, 00:08:34, 01:23:15

## Display Examples

### Example 1: Short Duration

**Visual:** `00:00:45`

**Description:**
- Duration: 45 seconds
- Format: HH:MM:SS

---

### Example 2: Medium Duration

**Visual:** `00:08:34`

**Description:**
- Duration: 8 minutes and 34 seconds
- Format: HH:MM:SS

---

### Example 3: Long Duration

**Visual:** `01:23:15`

**Description:**
- Duration: 1 hour, 23 minutes, and 15 seconds
- Format: HH:MM:SS
