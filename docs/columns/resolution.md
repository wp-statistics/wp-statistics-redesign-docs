# Resolution Column

Shows the screen resolution in width×height format.

## Column Configuration

- **Type**: Column (Reusable table column)
- **Status**: Done
- **Figma Design**: [Add link when available]

## Used In Widgets

This column is used in the following widgets:

- [Screen Resolutions Table](../widgets/screen-resolutions-table.md)

## Display

**Label:** Resolution

**Format:** "width×height" (e.g., "1920×1080")

**Alignment:** Left

**Examples:** 1920×1080, 1366×768, 2560×1440, 1440×900, 375×667

## Sortable Behavior

- **Sortable:** Yes
- **Default Sort Order:** Descending (by total pixels)
- **Sort Type:** Numeric (width × height)
