# Screen Resolution

The display dimensions of a visitor's screen in pixels (width × height).

## Definition

Screen resolution indicates the number of pixels displayed horizontally and vertically on a visitor's device screen.

**Format:** `Width × Height` (e.g., 1920×1080)

**Common Resolutions:**
- Desktop: 1920×1080, 2560×1440, 3840×2160
- Tablet: 1024×768, 2048×1536
- Mobile: 375×667, 414×896, 390×844
