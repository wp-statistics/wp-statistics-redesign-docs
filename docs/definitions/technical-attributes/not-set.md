# (Not set)

Placeholder value when a metric cannot be identified.

## Definition

Displayed in reports when specific data could not be captured or determined during tracking.

## When It Appears

- **Browser:** Unknown or rare user agent string
- **Operating System:** Undetectable from user agent
- **Referrer:** Blocked, stripped, or missing referrer header
- **Location:** GeoIP lookup failed
- **Device Type:** Could not determine from user agent

## Example

```
Browser Report:
Chrome        2,500 visitors (50%)
Firefox         800 visitors (16%)
Safari          700 visitors (14%)
Edge            500 visitors (10%)
(Not set)        50 visitors (1%)
```
