# 404 Page

HTTP error when requested URL doesn't exist.

## Common Causes

- Broken links (external sites, old URLs)
- Typos in URL
- Deleted content without redirects
- URL structure changes
- Moved content without 301 redirects

## What WP Statistics Tracks

- URL that returned 404
- Referrer (where broken link came from)
- Frequency
- Date/time

## Common Misconceptions

**"All 404s are bad"**
- No. Some are normal (typos, spam bots). Focus on high-traffic 404s.
