# Summary Totals

Aggregated metric values displayed as headline numbers.

## Definition

Summary totals are the combined counts or calculated values shown at the top of reports and dashboards. They provide a quick overview of key metrics for a selected date range.

## Common Summary Totals

- **Total Visitors:** Sum of unique visitors
- **Total Views:** Sum of all page views
- **Total Sessions:** Sum of all sessions
- **Average Session Duration:** Mean duration across sessions
- **Bounce Rate:** Overall percentage of single-page sessions

## Example Display

```
┌─────────────┬─────────────┬─────────────┐
│  Visitors   │    Views    │  Sessions   │
│   12,450    │   45,230    │   18,750    │
└─────────────┴─────────────┴─────────────┘
```

## Calculation Period

Summary totals are calculated based on the selected date range filter (Today, Last 7 Days, This Month, Custom Range, etc.).
